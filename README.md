# GourmetCamp

GourmetCamp は、グルメ情報サイトを想定して制作した Web サイトのトップページデザインです。  
エリア・ジャンル・タグなど、ユーザーが目的の飲食店を探しやすい構成で設計しています。

## デモ
https://iris-848.github.io/GourmetCamp/

## 動作環境
- Google Chrome / Microsoft Edge / Safari で動作確認済み

## 使用技術
- HTML
- CSS

※ 本プロジェクトは静的サイトのため、JavaScript は使用していません。

## 特徴
- エリア別・スポット別に探せるナビゲーション構成
- 検索フォーム（エリア・キーワード入力）
- ジャンル・タグから直感的に選べるUI
